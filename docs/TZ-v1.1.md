# «Альбом Мышления» — Техническое задание v1.1

> Актуально на: июнь 2025 · Пилот: ~20 детей, 10 лет, срок 1–2 месяца

---

## 1. Продукт

**«Альбом Мышления»** — геймифицированная платформа по логике и олимпиадной математике.

| Параметр | Значение |
|----------|----------|
| Аудитория | Дети ~10 лет |
| Пилот | ~20 детей |
| Фокус | **10 типов мышления**, внутри — темы-ветки |
| Контент | Все **40 веток** на карте; открываются **постепенно** по мере наполнения |
| Язык | Русский |
| Устройства | Tablet-first + desktop + mobile |
| Дизайн | По PNG-макетам (Figma нет); иллюстрации — генерация по запросам |
| Кристаллы | **Не в MVP** (убрать из UI) |

## 4. Задачи — прогрессия внутри темы

| Этап | Задачи | Формат | Макс. ★ |
|------|--------|--------|---------|
| 1. Проводник | 1–3 | Пошаговый интерактив (вопросы → выбор → авто-текст) | 3 |
| 2. Конструктор | 4–15 | `{число} {знак} {число} = {число}` + текст с пропусками | 4 |
| 3. Шаблон | 16–40 | Меньше подсказок, тот же шаблон ответа | 4 |
| 4. Самостоятельность | 41–50 | Полное решение + свой текст; 5★ — с листочком | 5 |

### Звёзды за задачу (не Daily)

| ★ | Условие |
|---|---------|
| 0 | Не открывал / ничего не сделал |
| 1 | Пробовал, не решил или неверно |
| 2 | Был ход, но кривой → неверный ответ |
| 3 | Верно, без объяснения |
| 4 | Верно с объяснением |
| 5 | Верно + объяснение на листочке + **проверка админом** |

**5★:** загрузка фото → статус «На проверке» → админ одобряет/отклоняет → только тогда 5★. Доверительного начисления нет.

### Типы блоков задачи (интерактив)

- `drag_select` — выбор участников
- `table_input` — таблица с полями
- `number_input` — числовой ответ
- `text_input` — текстовый ответ (этапы 3–4)
- `single_choice` / `multi_choice`
- `auto_explanation` — сборка текста решения

**Эталон:** задача №1 «Цыплята и змеи» — 9 шагов.

## 5. Видео — трёхуровневая модель (v1.1)

> **Принцип:** видео не дублирует интерактив, а дополняет его. Интерактив — основной путь; видео — intro темы и помощь по SOS.

### 5.1. Уровни видео

| Уровень | Назначение | Когда показывается | Кол-во на пилот |
|---------|------------|-------------------|-----------------|
| **Intro темы** | О чём ветка, зачем метод, один классический пример | Первый вход в тему (можно пропустить) | **1 на тему** (~12 для активных веток) |
| **Интерактив в задаче** | Пошаговое обучение | Всегда — основной путь | Заменяет видео на этапах 1–2 |
| **Видео-помощь** | Если ребёнок застрял | Только по кнопке «Объяснение» | 0 на этапах 1–2; точечно на 3–4 |
| **Разбор после** | Закрепление | После сдачи или при ≤2★ | Опционально, без автоплея |

### 5.2. Чего не делать (перегруз UI)

- ❌ Автоплей перед каждой задачей
- ❌ Отдельный плеер на каждой из ~50 задач темы
- ❌ Большой плеер на главной
- ❌ Обязательное «досмотри до конца» перед решением

### 5.3. UI — intro темы

При первом входе в тему:

```
┌─────────────────────────────────────┐
│ 🎬 Новая тема: Головы и ноги         │
│ [ превью 16:9 ]                      │
│ Посмотри ~5 мин — потом решим вместе │
│ [ Смотреть ]  [ Пропустить → ]       │
└─────────────────────────────────────┘
```

- «Пропустить» — для повторного входа / знакомых с темой
- Повторный просмотр — через «🎬 О теме» на странице ветки, не блокирует задачи

### 5.4. UI — помощь в задаче

Один блок помощи, не раздувая экран:

```
[ 💡 Подсказка ]   [ 🎬 Объяснение ]
```

| Кнопка | Содержание |
|--------|------------|
| Подсказка | Текст / один следующий шаг |
| Объяснение | Короткое видео **или** «медленный режим» (те же шаги + больше наводок) |

- Плеер — **модальное окно или overlay**, не встроен в layout задачи
- После 2–3 ошибок — мягкое предложение открыть «Объяснение»

### 5.5. Прогрессия «меньше видео к концу темы»

| Этап задач | Видео |
|------------|-------|
| 1–15 | Только intro темы (1 раз) |
| 16–40 | Кнопка «Объяснение» по желанию |
| 41–50 | Короткий разбор доступен после попытки или при низкой оценке |

### 5.6. Влияние на звёзды

| Действие | Влияние |
|----------|---------|
| Просмотр intro темы | Без штрафа |
| Открытие «Объяснение» в задаче | Потолок **3★** (не 4–5) |
| Прохождение без видео-помощи | Полный диапазон ★ по правилам этапа |

### 5.7. Объём видео на пилот (рекомендация)

```
1 intro × активная тема          (~12 роликов)
+ 0 видео на задачи этапа 1–2    (интерактив достаточен)
+ 1 «метод темы» опционально     (можно объединить с intro)
+ 1–3 точечных на самые сложные  (задачи 41–50)
```

**Не цель пилота:** 50 × 12 = 600+ роликов на задачу.

### 5.8. Хостинг и данные

| Поле (тема) | Описание |
|-------------|----------|
| `introVideoUrl` | URL (YouTube / Vimeo / mp4) |
| `introVideoDuration` | минуты (для UI) |
| `methodVideoUrl` | опционально, если отделено от intro |

| Поле (задача) | Описание |
|---------------|----------|
| `helpVideoUrl` | опционально, не у всех задач |
| `slowModeSteps` | альтернатива видео — расширенные подсказки в тех же шагах |

| Событие в логе | Для родителей / ИИ |
|----------------|-------------------|
| `intro_watched` / `intro_skipped` | — |
| `help_video_opened` | «3 раза смотрел объяснение в теме X» |

### 5.9. Фазировка реализации видео

| Фаза | Что |
|------|-----|
| **B** | `introVideoUrl` + модалка на странице темы + пропуск |
| **B** | Кнопка «🎬 О теме» для повтора |
| **C** | `helpVideoUrl` + учёт для потолка ★ |
| **D** | «Медленный режим» без отдельного ролика |
| **После пилота** | Загрузка видео из админки |

## 6. Daily (Пн–Пт) и раздел «Задачи»

### Два контура (не смешивать)

| Контур | Меню / раздел | Содержание |
|--------|---------------|------------|
| **Daily** | «Задачи» в sidebar | Чтение, русский, школьная математика |
| **МышМат** | Блок на `/tasks` + карта тем | Олимпиадные темы («Головы и ноги» и др.) |

**Математика daily ≠ олимпиадный МышМат.** Daily-математика не ведёт на «Головы и ноги».

### Раздел `/tasks` — состояния экрана

| Ситуация | UI |
|----------|-----|
| Пн–Пт, daily не завершён | Карточки: чтение, русский, математика |
| Пн–Пт, всё daily сделано | «Ты молодец!» + блок **МышМат** для тренировки |
| Сб–Вс (выходной) | «Сегодня выходной» + **МышМат** без daily |

### Полоска Пн–Пт на главной

- Клик → переход в `/tasks`
- Галочки — реально выполненные daily-дни текущей недели
- Новая неделя → полоска сбрасывается
- Новый день → сброс «сегодняшних» предметов

### Звёзды за daily

### Состав дня (все предметы в MVP)

| Предмет | Задач/день |
|---------|------------|
| Чтение | 1 |
| Русский язык | 2–3 |
| Математика | 3–5 |

### Звёзды за неделю

| День | ★ | Накопительно |
|------|---|--------------|
| Пн | 1 | 1 |
| Вт | 2 | 3 |
| Ср | 3 | 6 |
| Чт | 4 | 10 |
| Пт | 5 | **15** |
| Бонус за идеальную неделю | | **+25** |
| **Максимум** | | **40★/нед** |

- Пропуск дня → бонус 25★ не даётся
- **Волшебный пендаль:** 50★, спасает 1 пропущенный день
- Пендаль можно получить за неделю без пропусков

## 8. Карточки альбома

- Заряд: 5% → 8% → 12% → 17% → 23% → 30% (после 1–6 задач по теме)
- UI: «Карточка заряжена на N%»
- Повторы → копии → апгрейд (3 об → серебро → 3 сер → золото → редкая)
- Редкости: обычная, серебряная, золотая, редкая, дуэльная, бумажная
